# Performance Data

This folder will contain winning ad copy with performance metrics.

---

## Coming Soon

Add files here with:
- Top performing Meta ads (copy + metrics)
- Winning email subject lines + open rates
- SMS campaigns + click rates
- A/B test learnings

## Suggested Format

For each winning piece of copy, include:

```
### [Campaign Name]
**Channel:** Meta / Email / SMS
**Date:** [When it ran]

**Copy:**
[The actual copy]

**Metrics:**
- CTR: X%
- ROAS: X.X
- Conversion Rate: X%

**Why It Worked:**
[Your hypothesis on what made it perform]
```

## How This Will Be Used

When generating new copy, the AI will reference top performers to:
- Match proven structures
- Use similar hooks and angles
- Learn from what resonates with your audience
