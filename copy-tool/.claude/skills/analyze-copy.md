# Analyze Copy Performance Skill

When analyzing copy performance for Jones Road:

## How to Analyze Winning Copy

Break down each piece by:

1. **Hook** - What's the opening line? Why does it stop the scroll?
2. **Structure** - How is it organized? (Problem â†’ Solution, Benefit-first, Story, etc.)
3. **Voice** - How does it sound? (Conversational, authoritative, urgent, etc.)
4. **Proof** - What credibility elements? (Reviews, ingredients, founder story, etc.)
5. **CTA** - What's the call to action? How strong?

## What to Look For Across Multiple Winners

- Repeated phrases or hooks
- Common structures
- Consistent tone patterns
- Which products/benefits appear most
- Length patterns (short vs long)

## Compare Against Existing Frameworks

Reference the training data at `training-data/channels/` to see:
- What patterns are already documented
- What's missing that should be added
- What's documented but not appearing in winners (might need to remove)

## Output Format

When sharing analysis, use this structure:

### Patterns Found
- [List the 3-5 biggest patterns]

### Recommended Framework Updates
- [Specific changes to make to training data]

### Example Additions
- [Which specific examples should be added, and where]

## After Analysis

Offer to:
1. Add winning examples to the relevant training data file
2. Update frameworks based on patterns found
3. Create a branch with all changes for review
